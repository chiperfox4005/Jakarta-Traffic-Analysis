# Data Mining Proyek - Kemacetan di DKI Jakarta

## Deskripsi
Proyek ini merupakan analisis data mining terkait **kemacetan di DKI Jakarta**.  
Tujuan proyek adalah untuk mengolah data, menemukan pola kemacetan, serta memberikan insight berdasarkan hasil analisis.

## Tujuan Proyek
- Mengumpulkan data terkait kemacetan di Jakarta
- Membersihkan dan memproses data
- Melakukan analisis data mining
- Menampilkan hasil analisis dalam bentuk visualisasi

## Tools yang Digunakan
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / VS Code
