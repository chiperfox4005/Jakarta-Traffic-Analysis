# Jakarta Traffic Analysis

## Overview

Jakarta Traffic Analysis merupakan proyek data mining yang berfokus pada analisis pola kemacetan di DKI Jakarta melalui proses pengolahan dan eksplorasi data.

Proyek ini bertujuan untuk menghasilkan insight berbasis data mengenai kondisi lalu lintas, mengidentifikasi pola kemacetan, serta menyajikan hasil analisis dalam bentuk visualisasi yang informatif untuk mendukung pemahaman terhadap dinamika mobilitas perkotaan.

---

## Project Objectives

* Mengumpulkan dan menyiapkan dataset terkait kondisi lalu lintas di DKI Jakarta
* Melakukan data cleaning dan preprocessing
* Menganalisis pola kemacetan menggunakan pendekatan data mining
* Menghasilkan visualisasi untuk interpretasi data
* Menyusun insight berdasarkan hasil analisis
